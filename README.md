# Resonance MIDI Player

「ブループロトコル：スターレゾナンス」の楽器演奏向けに、Standard MIDI File（MIDI）の音符をWindowsのキーボード入力へ変換するデスクトップツールです。

ゲームプロセスやメモリの読み書きは行わず、Windowsの通常のキー入力APIを使用します。

> [!IMPORTANT]
> **本ツールは必ず「管理者として実行」してください。**<br>
> `ResonanceMidiPlayer.exe`を右クリックし、「管理者として実行」を選択します。管理者権限がない場合、ゲームへキー入力が届かず演奏できないことがあります。

> 利用者は、ゲーム内に表示される楽器演奏機能の確認事項へ承諾済みであり、自作曲、権利者から許諾を得た曲、または利用条件上演奏可能な曲だけを使用するものとします。権利侵害や利用規約違反にあたる演奏は、アカウント停止（BAN）などの措置を受ける可能性があります。
>

## v0.0.3の既知の制限

現在対応している楽器は**キーボード（ピアノ）のみ**です。ギター・ベース・ドラムは音域とキー配置が異なるため未対応で、今後のバージョンアップで対応する予定です。

作者がゲーム内の音域解放に必要な実績をまだ達成していないため、現在確認・対応できている音域は**C3～B6の4オクターブ**です。それより低い音・高い音には未対応です。

ゲーム内で追加音域を検証できるようになり次第、今後のバージョンアップで対応する予定です。

音域切替はゲームへ`L Shift`・`L Ctrl`・`>`・`<`のキー入力を送って行います。短時間に音域切替が集中する曲や、ゲーム側の処理負荷が高い状況では、切替入力が認識されず、**ツールが想定する音域とゲーム画面の鍵盤位置がずれる**場合があります。その場合は演奏を停止し、ゲーム側の鍵盤を`Z = C3`の初期位置へ戻してから再生してください。

## 主な機能

- SMF format 0 / 1の`.mid`・`.midi`ファイルを読み込み
- 再生・停止・先頭へ戻る・シーク
- 再生速度、移調、キー保持時間、開始カウントダウンの設定
- MIDI channel 10のドラム除外
- 再生前のサステイン状態チェック（初期値オン）
- C3～B6の4オクターブ演奏
- 譜面全体を先読みし、`L Shift`・`L Ctrl`・`>` / `<`の有効条件を追跡する音域表示の自動切り替え
- 対象ゲームウィンドウの自動再取得とフォーカス
- 常に手前、背景透明度、UIサイズ（0.5～2.0倍）、ウィンドウ位置の保存

## ダウンロード

GitHubのReleasesから最新版のZIPをダウンロードし、展開後に`ResonanceMidiPlayer.exe`を実行してください。

デジタル署名のない個人制作アプリのため、Windows SmartScreenが警告を表示する場合があります。

## 基本的な使い方

1. `ResonanceMidiPlayer.exe`を右クリックし、「管理者として実行」を選択します。
2. ゲームでピアノの演奏画面を開きます。
3. `+ MIDI`から演奏するMIDIファイルを選択します。
4. 対象欄で「ブループロトコル：スターレゾナンス」を選択します。
5. 再生ボタンを押します。

ゲーム側の鍵盤初期位置は`Z = C3`を前提としています。4オクターブ自動切り替えを有効にすると、再生前に譜面全体を解析します。低音・初期・高音の3つのベース位置と、`L Ctrl`・補正なし・`L Shift`の状態を別々に追跡し、ゲーム側で無効になる操作を避けた切替経路を先に決定します。`L Shift`と`L Ctrl`を直接切り替えた場合は2オクターブ移動として扱います。同時刻の音が1つの3オクターブ表示に収まらない場合だけ、高音側を素早く発音して元の位置へ戻します。シーク後は開始カウントダウン中に、次の打鍵に必要な表示位置へ準備します。

初期設定は次のとおりです。

- 再生速度：1.0
- 移調：0半音
- キー保持：1ms
- 音域切替待機：20ms
- ドラム除外：オン
- 4オクターブ自動切替：オン
- 開始カウントダウン：3秒
- サステイン状態のチェック：オン
- 常に手前：オン
- 背景透明度：80%
- UIの大きさ：1.0倍

設定は`%APPDATA%\ResonanceMidiPlayer\config.json`に保存されます。リポジトリや配布ZIPにユーザー設定は含まれません。

## ソースから実行

WindowsとPython 3.11以降を使用してください。

```powershell
python -m pip install -r requirements.txt
python qt_app.py
```

## EXEをビルド

PyInstallerをインストールしてから実行します。

```powershell
python -m pip install pyinstaller
.\build.ps1
```

完成物は`dist\ResonanceMidiPlayer.exe`です。

## テスト

```powershell
python -m unittest discover -s tests -v
```
