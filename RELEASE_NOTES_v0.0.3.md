# Resonance MIDI Player v0.0.3

## 主な変更

- 譜面全体を先読みし、`L Shift`・`L Ctrl`・`>`・`<`を組み合わせる音域切替計画を追加
- 左Shift／左Ctrlのトグルと、3オクターブ単位の鍵盤移動ルールに対応
- 休符中の音域切替と、シーク後の開始カウントダウン中の事前位置合わせに対応
- C#5とD#5の割り当てを、それぞれ`I`と`O`へ修正
- 既存の誤った黒鍵設定を起動時に自動移行
- 遅延後も打鍵が短くなりすぎないよう、実際のキー保持時間を補正

## 既知の制限

- 現在対応している楽器はキーボード（ピアノ）のみです。ギター・ベース・ドラムは音域とキー配置が異なるため未対応で、今後対応する予定です。
- 作者がゲーム内の音域解放実績をまだ達成していないため、確認・対応済みの音域はC3～B6です。
- 音域切替はゲームへの疑似キー入力で行います。短時間に切替が集中する曲やゲーム側の処理負荷が高い状況では、切替入力が取りこぼされ、ツール内部の想定とゲーム画面の鍵盤位置がずれる場合があります。
- 鍵盤位置がずれた場合は演奏を停止し、ゲーム側を`Z = C3`の初期位置へ戻してから再生してください。

本バージョンは音域切替方式を検証するための初期リリースです。今後のバージョンで安定性を改善する予定です。
