# Resonance MIDI Player v0.1.0

## 主な変更

- 音域補正（ベータ）を追加。演奏できる音域の外にある音を、オクターブ単位で音域内へ移動して鳴らせるようになりました
- 音域の解放段階を設定できるようになりました。実績で解放した範囲だけを演奏対象として扱います
- ヘッダーに音符アイコンのボタンを追加。音域補正の設定をここから開きます
- 解放していない音域の音のために鍵盤が動かないよう、音域切替の計画を修正

## 音域補正について

ゲーム内の実績で解放していない音域は、鍵盤が灰色で表示され音が鳴りません。
そのため解放が進んでいない状態では、低音や高音を含む曲を読み込んでも該当する音が抜け落ちます。

音域補正を有効にすると、演奏できない音をオクターブ単位で移動させ、鳴らせる範囲へ収めます。
音符ごとに個別に折り返すとメロディの起伏が崩れるため、本ツールでは**フレーズ単位でまとめて移動**します。
和音は分解せず、同時に鳴る音はまとめて同じ量だけ移動します。

設定は音符アイコンのボタンから開きます。「音域外の音を…補正する」にチェックを入れ、
「音域の解放段階」を自分の解放状況に合わせて選んでください。

- 初期状態：C3～B4
- 実績①解放後：C3～B6
- 実績②解放後：A0～B6
- 実績③解放後：A0～C8

チェックを入れていない場合は、これまでと同じく音域外の音は鳴りません。

## 既知の制限

- 音域補正はベータ版です。フレーズ単位で移動する方式のため、原曲どおりの音の高さにはなりません。特に演奏できる音域が狭い状態（初期状態など）では、移動が頻繁に発生して原曲の印象が変わる場合があります。
- 音域補正は演奏できる範囲へ収めることを優先します。1つのトラックにメロディと伴奏が混在している曲では、意図と異なる移動になる場合があります。
- 現在対応している楽器はキーボード（ピアノ）のみです。ギター・ベース・ドラムは音域が異なるため未対応で、今後対応する予定です。
- 作者がゲーム内の音域解放実績をまだ達成していないため、実際に発音させて確認できているのはC3～B6の4オクターブです。それより低い音・高い音を含む曲は未検証です。
- 音域切替はゲームへの疑似キー入力で行います。短時間に切替が集中する曲やゲーム側の処理負荷が高い状況では、切替入力が取りこぼされ、ツール内部の想定とゲーム画面の鍵盤位置がずれる場合があります。
- 鍵盤位置がずれた場合は演奏を停止し、ゲーム側を`Z = C3`の初期位置へ戻してから再生してください。

音域補正は、音域解放が進んでいない環境でも曲を演奏できるようにするための機能です。今後のバージョンで移動のしかたを調整する予定です。
